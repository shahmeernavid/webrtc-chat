Peer to peer chat application

# Running Locally

To run on your local machine:

1. Create a virtual environment and start it:

    ```
    cd chat
    virtualenv env
    . env/bin/activate
    ```

2. Install python dependencies:

    ```
    pip install -r requirements.txt
    ```

3. Install node dependencies:

    ```
    npm i
    ```

4. Running the app

    ```
    gulp start
    ```
